## 码率设置

!> 什么是码率

通俗地讲，就是直播画面每秒改变的像素数量，码率越高，改变量越多，画面越精确；当然占用的网速也就越高。


!> 推荐码率设置


| 画质 | 视频分辨率 | H.264转码码率 | 
| ---- | ---- | ---- | 
| 标清（480P） | 854*480 | 600Kbps |
| 高清（720P） | 1080*720 | 1000Kbps |
| 超清（1080P） | 1920*1080 | 2000Kbps |


!> 码率设置步骤


1: 打开OBS，点击主界面右下方【控件】-【设置】 - 【输出】，进入设置界面
